# Repository Label Colors

Unified label colors applied across all private repositories. Programming language colors follow the
official [GitHub Linguist](https://github.com/github-linguist/linguist/blob/main/lib/linguist/languages.yml) palette.
Default issue labels follow GitHub's stock colors.
